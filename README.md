## 目录结构

- \_static 开发用的静态页面
- config 配置信息
- libs 一些工具库
- models 模型（M）
- public 公共静态文件
- routes 路由（C）
- views 视图（V）
- package.json
- server.js
- .env 环境变量配置文件(不要上传，避免账号泄露)

## 主要技术栈

- express
- mysql
- ejs
